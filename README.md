# Hierarchical Clustering for Pattern Discovery using Iris Dataset

# Summary

This project implements Agglomerative Hierarchical Clustering on the Iris dataset using four different linkage methods:

## Single Linkage
## Complete Linkage
## Average Linkage
## Ward Linkage

The project compares clustering performance using dendrogram analysis, silhouette scores, PCA-based visualization, and distance matrix heatmaps. The objective is to analyze how different linkage strategies affect cluster formation and clustering quality.
